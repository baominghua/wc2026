# 2026美加墨世界杯 · 预测与数据分析网站 — 产品需求文档（PRD）

---

## 一、产品概述

### 1.1 产品名称
**WorldCup Lens · 世界杯透视镜**

### 1.2 产品定位
面向全球足球爱好者、体育媒体从业者及竞猜用户的一站式世界杯数据洞察与赛事预测平台，以数据驱动为核心，提供从小组赛到决赛的全赛程深度分析与智能预测。

### 1.3 背景与机会
- **2026美加墨世界杯**是历史上首次扩军至48支球队的世界杯，12个小组、104场比赛，赛事规模与数据量均为历届之最
- 比赛时间对亚洲观众友好（约70%的比赛在北京时间6:00-11:00），中国用户参与度预期极高
- 赛制变革（新增32强淘汰赛阶段）带来全新的分析维度与预测空间
- 市面上缺乏针对48队赛制的深度数据预测产品

### 1.4 目标用户

| 用户角色 | 核心诉求 | 典型行为 |
|---------|---------|---------|
| **休闲球迷** | 快速了解赛况、看预测结果 | 浏览赛事日历、查看胜负预测 |
| **硬核球迷** | 深度数据分析、战术洞察 | 查看球员数据、球队对比、历史战绩 |
| **竞猜用户** | 高质量预测参考、奖金参考分析 | 关注AI预测、市场参考变化、专家观点 |
| **体育媒体** | 数据素材、可视化图表 | 导出数据、嵌入图表、引用分析结论 |
| **社区用户** | 互动讨论、预测PK | 参与竞猜、发帖讨论、排行对比 |

## 二、核心功能模块

### 2.1 赛事全景 🏟️

#### 2.1.1 赛事日历
- 按日期/小组/阶段筛选的完整104场赛程表
- 比赛时间自动转换为用户本地时区
- 赛前提醒订阅（浏览器通知 / 微信推送）
- 16座主办球场信息（位置、容量、历史）

#### 2.1.2 12组小组赛看板
- 12个小组的实时积分榜
- 小组出线概率模拟（蒙特卡洛模拟10万次）
- 第三名最佳8支出线规则可视化说明
- 小组末轮多种出线情景推演

#### 2.1.3 淘汰赛对阵树
- 从32强到冠军的完整对阵图
- 实时更新已确定对阵
- 点击任意节点查看预测胜率

### 2.2 智能预测 🤖

#### 2.2.1 AI赛前预测
- **综合预测模型**：融合Elo等级分、xG数据、球员状态、主客场因素的集成学习模型
- 每场比赛输出：胜/平/负概率、预测比分、预测总进球数
- 模型置信度可视化（高/中/低）
- 历史预测准确率追踪

#### 2.2.2 赛中实时预测
- 接入实时赛况API，动态调整预测概率
- 关键事件（进球/红牌/换人）触发概率跳变
- 实时胜率曲线图

#### 2.2.3 冠军路线模拟
- 基于当前赛况模拟各队夺冠概率
- 最可能对阵路线图
- "黑马路径"分析——低排名球队的突破概率

### 2.3 数据中心 📊

#### 2.3.1 球队画像
- 48支球队完整数据档案
- 攻防雷达图（进球/失球/控球率/传球成功率/射门/抢断）
- 历届世界杯战绩时间线
- 世界杯预选赛表现回顾
- 关键球员列表与数据

#### 2.3.2 球员数据库
- 核心球员详细数据（出场/进球/助攻/评分）
- 球员对比工具（双人/多人雷达图叠加）
- 球员身价与俱乐部信息
- 伤病/停赛状态追踪

#### 2.3.3 对阵分析
- 任意两队历史交锋记录
- 战术风格匹配度分析
- 关键对位分析（如：某队左路进攻 vs 对手右路防守）
- 相似对手参照（"A队类似于B队，而C队曾击败B队"）

#### 2.3.4 市场参考追踪
- 合规数据源市场参考变化时间线
- 外部共识隐含概率 vs 模型预测概率对比
- 市场参考异常波动预警
- 高偏差参考识别（模型概率显著高于外部共识隐含概率时标记）

### 2.4 竞猜社区 🏆

#### 2.4.1 预测竞猜
- 每场比赛可提交自己的胜负预测
- 比分竞猜（猜对比得3分，猜对胜负得1分）
- 累计积分排行榜
- 好友PK / 房间制竞猜

#### 2.4.2 专家专栏
- 邀请足球分析师/前职业球员入驻
- 发布赛前分析长文
- 专家预测记录与准确率追踪

#### 2.4.3 话题广场
- 每场比赛自动创建讨论帖
- 热门话题（如"谁是最强黑马""最佳射手预测"）
- 支持图文/投票/引用数据卡片

### 2.5 可视化工具 📈

#### 2.5.1 数据图表库
- 预制20+可视化模板（雷达图/热力图/桑基图/树状图等）
- 所有图表支持一键导出为PNG/SVG
- 媒体引用嵌入代码生成

#### 2.5.2 自定义对比
- 用户自选维度进行球队/球员对比
- 拖拽式图表配置
- 分享自定义图表链接

## 五、技术要点

### 5.1 数据源
- **赛况数据**：FIFA官方API / Opta / SportRadar
- **历史数据**：Kaggle足球数据集 / 俱乐部赛季数据（Transfermarkt）
- **外部市场参考数据**：合规 API / 授权数据源
- **球员数据**：FBref / WhoScored / SofaScore

### 5.2 预测模型
- 基础模型：Elo等级分 + 泊松进球模型
- 增强模型：XGBoost集成（融合球员状态/伤病/主客场/天气等特征）
- 实时模型：赛中贝叶斯更新
- 蒙特卡洛模拟：10万次赛程模拟，输出各阶段晋级/夺冠概率

#### 5.2.1 当前实现：NA-ATLAS 26.3
- **核心链路**：Elo/FIFA排名/状态评分 → 主场与地域优势修正 → xG估算 → 0-8球泊松/Dixon-Coles比分矩阵 → 胜平负、比分、大小球、进球球员概率。
- **主场规则**：小组赛阶段美国、墨西哥、加拿大均按完整主场；淘汰赛阶段只有美国按完整主场；加拿大、墨西哥及其他美洲球队按半主场；双方均为美洲球队时半主场抵消。
- **外部因子**：天气、场地、高压逼抢、伤停、体能、强制中立均会进入xG和比分矩阵，不只改变说明文字。
- **红黄牌机制**：仅在官方实时事件或可信赛后统计接入后启用，避免使用不准确的硬编码黄牌累计；接入后会影响首发可用性、阵容置信度、防守侵略性和后续禁赛风险。
- **阵容来源口径**：内置候选阵容池，并预留 WhoScored probable lineup/team news、RotoWire predicted/confirmed lineups 与 injury table、SofaScore/FotMob 实时事件与技术统计的接入路径。
- **说明文档**：详见 `模型预测依据说明.md`。

### 5.3 性能要求
- 首页加载 < 2s
- 实时预测延迟 < 5s（赛中事件到概率更新的时间）
- 支持10万+并发用户
- 图表渲染流畅（60fps 交互）

### 5.4 多端适配
- 响应式Web（桌面端为主，移动端适配）
- PWA支持（离线赛程查看）
- 微信小程序（核心预测+赛程功能）

## 七、差异化竞争点

1. **48队赛制专项分析**——市面上首个针对新赛制出线规则（含第三名最佳8支）的深度模拟产品
2. **赛中实时概率跳变**——不只是比分播报，而是概率视角的实时叙事
3. **外部共识 vs 模型对比**——帮用户发现高偏差参考场景，而非简单给结论
4. **可分享的数据卡片**——一张图总结一场比赛的核心数据，社交媒体传播利器
5. **中文生态优先**——比赛时间友好+中文深度内容，抢占中文市场空白

## 需求补充：赛后复盘、公开数据源与下载

新增“复盘”模块，用于在比赛结束后自动对照赛前预测与真实赛果，输出单场复盘报告和整体预测准确率表。统计口径包含 1 选准确率、2 选准确率、3 选准确率、比分命中率和冷门参考命中率。单场报告和整体统计表均支持下载。

所有客观事实数据必须来自公开、免费或用户自行配置的合规来源。没有商用接口时，系统优先使用 ESPN 公共比分端点、FIFA 官方公开页面、football-data.org 免费额度或公开网页抓取。拿不到的数据明确显示未取得公开数据，不能用手工 JSON、样例数据或预测值冒充事实数据。

详细需求补充见 `docs/需求补充-赛后复盘与公开数据约束.md`。
