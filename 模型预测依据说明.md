# 美加墨世界杯AI预测模型 26.5 预测依据说明

本文档说明预测页使用的分析依据、数据边界和 26.5 版本新增的外部市场信号、冷门参考逻辑。模型只负责预测与解释；赛果、积分、射手榜、助攻榜、红黄牌、停赛、阵容、观众人数等客观数据必须由实时数据源自动同步，不应把预测值写入统计模块。

## 1. 模型输入总览

模型会把输入拆成四类：球队实力、比赛事实、场景环境、外部共识。每一类只影响预测，不替代真实赛果。

### 球队实力

- Elo 强弱分、FIFA 排名、洲际强度修正。
- 最近比赛状态、进攻 xG、防守 xGA、进球效率、失球稳定性。
- 主力阵容完整度、关键球员可用性、门将/中轴线/锋线质量。
- 球队风格：控球、反击、高位压迫、边路推进、定位球、防守反击、低位防守等。

### 比赛事实

- 赛事阶段：小组赛、淘汰赛、加时和点球风险。
- 小组积分形势、净胜球压力、出线动机。
- 已发生的赛果、进球、助攻、黄牌、红牌、停赛、伤病和首发信息。
- 已完成比赛的真实比分、技术统计、观众人数和事件时间线。

### 场景环境

- 主场、半主场、中立场。美国、加拿大、墨西哥按赛事阶段自动判定；其他美洲球队获得半主场氛围加成。
- 天气、温度、降雨、风速、场地类型和比赛地时区。
- 旅途负荷、连续作战、短休、跨时区和高强度逼抢消耗。
- 战术与阵型相克：如高位压迫对出球能力、三中卫对边路宽度、低位防守对传中质量、定位球优势对密集防守。

### 外部共识

- 可选接入赛前市场信号，用于观察外部共识与模型分歧。
- 外部信号只做轻量校准，不直接覆盖模型。
- 未配置、请求失败、没有匹配赛事或没有相关市场时，模型保持原始预测，只显示未接入状态。

## 2. 核心预测流程

1. 基础实力层：综合 Elo、FIFA、近期状态、攻防 xG、阵容完整度和赛程负荷，得到双方基础强弱差。
2. 场景修正层：合并主场/半主场、天气、场地、伤停、红黄牌停赛、体能、战术相克和比赛动机。
3. 进球矩阵层：根据双方预期进球生成 0-8 球比分概率矩阵，并使用低比分相关修正降低随机漂移。
4. 外部共识层：如果接入赛前市场信号，则对比胜平负方向、总进球倾向和让步强度。
5. 输出层：生成主预测比分、另外两个高概率比分、总进球与大小球倾向、胜平负概率、关键因子、冷门概率与冷门比分参考。

## 3. 外部市场信号权重

外部市场信号默认关闭，必须显式配置数据源和 API Key。配置变量沿用代码中的英文接口名，用户界面统一显示为“赛前市场信号”。

校准规则保持保守：

- 模型与外部共识最大分歧小于 6 个百分点：视为一致，不调整。
- 分歧 6-12 个百分点：外部信号权重 8%。
- 分歧 12-20 个百分点：外部信号权重 13%。
- 分歧超过 20 个百分点：外部信号权重 18%。

校准只轻度拉近胜平负概率。总进球外部信号会参与大小球倾向和代表比分选择，但不会覆盖比分矩阵。

## 4. 冷门概率

冷门参考用于提示“热门方向不稳”的风险，不等同于操作建议。它综合以下信号：

- 最终胜平负概率是否足够集中。
- 平局概率是否拖高，导致比赛更容易被单个事件改变。
- 外部共识是否与模型热门方向出现反向信号。
- 比分矩阵中低概率但合理的反向比分尾部概率。
- 主场、半主场、伤停、红黄牌、天气、体能和战术相克是否放大不确定性。

输出字段包括：

- `probability`：冷门概率。
- `risk_level`：低/中/高冷门风险。
- `score`：冷门比分参考。
- `score_probability`：该冷门比分在矩阵中的概率。
- `reasons`：冷门原因分析。

## 5. 图片导出内容

预测结果图片应包含：

- 主预测比分。
- 另外两个最可能比分。
- 胜平负概率。
- 总进球与大小球预测。
- 市场信号校准状态。
- 冷门概率、冷门比分和原因。
- 关键模型因子。

手机端下载图片优先使用浏览器原生下载能力；如果浏览器限制下载行为，页面应回退为打开图片数据供用户长按保存，不能跳回首页。

## 6. 数据边界

实时赛果、积分、射手榜、助攻榜、红黄牌、停赛和伤病属于客观事实数据，必须走自动同步数据源。预测模型不得把假数据、样例数据或预测值写入这些统计模块。

Opta、WhoScored、Stats Perform 等商业数据源涉及授权，项目只保留合规接口适配能力，不在代码中内置未授权抓取逻辑。
## 7. 赛后复盘与公开数据约束

赛后复盘模块会在比赛结束后，用真实赛果对照赛前预测，统计 1 选、2 选、3 选、比分和冷门参考命中率。复盘结论只作为后续预测的小幅校准，不覆盖 Elo、FIFA 排名、近期状态、主场、天气、阵容、牌面和战术相克等基础因子。

客观事实数据必须来自公开、免费或用户自行配置的合规来源。当前优先使用 ESPN 公共比分端点、FIFA 官方公开页面、football-data.org 免费额度或公开页面抓取。未取得公开数据时，页面和报告必须显示未取得公开数据，不能用预测值、手工 JSON 或样例数据补齐。

详细方案见 `docs/赛后复盘与公开数据源方案.md`。
