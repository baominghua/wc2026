# 赛后复盘与公开数据源方案

## 公开免费数据约束

当前项目不依赖商用接口。比分、赛程、射手榜、助攻榜、红黄牌、停赛、阵容、技术统计等客观事实，只允许来自公开、免费、可访问的数据源或用户自行配置的免费额度接口。

可用来源分层如下：

1. ESPN 公共足球比分端点：用于赛程、比分、比赛状态、部分事件。
2. FIFA 官方公开页面：用于官方排名、赛事信息和公开公告。
3. football-data.org 免费额度：可选，用于赛程与赛果备源。
4. 公开网页抓取：只抓公开页面，不绕过登录、付费墙、反爬或授权限制。

不使用未授权的 Opta、WhoScored、Stats Perform 商用数据抓取。后续如果拿到授权，可以新增适配器，但不能把未授权字段混入线上事实数据。

## 复盘模块

复盘模块用于回答三个问题：

- 赛前模型最看好的方向和实际结果差在哪里？
- 偏差来自进球效率、状态变化、阵容可用性、场景因素，还是小组形势？
- 下一轮预测应做什么轻量校准？

当前实现提供：

- `/api/v1/reviews/`：整体准确率与复盘行数据。
- `/api/v1/reviews/{match_id}`：单场复盘。
- `/api/v1/reviews/export.csv`：整体复盘统计下载。
- 前端 `/reviews` 页面：复盘看板、单场报告、下载入口。

## 准确率口径

- 1 选准确率：主预测胜平负命中。
- 2 选准确率：真实胜平负落在模型前两个方向内。
- 3 选准确率：真实胜平负落在模型三个方向内。
- 比分 1 选准确率：主预测比分完全命中。
- 比分 3 选准确率：真实比分落在主预测和另外两个高概率比分内。
- 冷门准确率：模型给出的冷门参考方向与真实结果一致。

## 复盘权重

复盘权重只做小幅调整：

- 同组已完赛球队状态明显上升时，下一场进攻预期轻微上调。
- 同组已完赛球队状态明显下降时，下一场进攻预期轻微下调。
- 若双方小组积分形势显示平局对双方都有较高收益，平局概率轻微上调。

这些权重不会覆盖 Elo、FIFA 排名、近期状态、主场、天气、阵容等基础因子。

## 降级策略

当公开源没有技术统计、助攻、角球或牌面数据时：

- 页面显示“未取得公开数据”。
- 复盘报告说明该字段未参与偏差判断。
- 模型不使用手工样例或预测值冒充事实数据。
